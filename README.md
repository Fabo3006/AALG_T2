1. Solicita al usuario el tamaño N de una matriz cuadrada NxN.
2. Genera una matriz de tamaño NxN.
3. Llena la matriz con números aleatorios entre 99 y 999.
4. Muestra la matriz en formato cuadrado.
5. Utiliza la estrategia Divide y Vencerás para contar los números múltiplos de 5 o 7.
6. Muestra la cantidad total de números múltiplos de 5 o 7 encontrados en la matriz.

Recordar que: La prueba documentada en el archivo `resultado.docx` fue realizada utilizando una matriz de tamaño 11x11.
